# Task Management Application

## Overview
A simple Task Management Application built using HTML, CSS, and JavaScript. This application allows users to add, view, and delete tasks easily.

## Features
- Add new tasks
- View task list
- Delete tasks by clicking on them
- Simple and user-friendly interface
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure

```
task-management-app/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it in a web browser.
4. Start adding and managing tasks.

## Usage

1. Enter a task in the input field.
2. Click the **Add Task** button.
3. The task will appear in the list.
4. Click on a task to remove it.
